# LexicAZ

LexicAZ es una aplicación móvil desarrollada en **Unity** cuyo objetivo es ayudar a aprender vocabulario en varios idiomas de forma simultánea y progresiva mediante distintos minijuegos.

Este proyecto ha sido desarrollado como **Proyecto Final del Grado Superior DAM (Desarrollo de Aplicaciones Multiplataforma)**.

---

##  Objetivo del proyecto

Facilitar el aprendizaje de vocabulario comparando una misma palabra en varios idiomas, permitiendo al usuario:
- Seleccionar categorías de palabras
- Elegir y ordenar los idiomas de aprendizaje
- Aprender mediante minijuegos interactivos
- Practicar y reforzar vocabulario de forma progresiva

---

##  Funcionalidades principales

-  Selección de categorías de vocabulario
-  Configuración del orden de idiomas
-  Minijuegos:
  - **Encuentra la mentira**
  - **Encuentra la verdad**
  - **Escribe la palabra**
  - **Repaso**
-  Base de datos local SQLite (funciona sin conexión)
-  Interfaz adaptada a distintos tamaños de móvil

---

##  Tecnologías utilizadas

- **Unity** (C#)

- **Android**
- **Git & GitHub**

- **Ubuntu Linux**

---

##  Cómo probar la aplicación

La aplicación está disponible en formato **APK** para Android.

1. Descarga el archivo `LexicAZ.apk`
2. Instálalo en un dispositivo Android
3. Acepta la instalación desde orígenes desconocidos si es necesario

>  Proyecto académico – no publicado (de momento) en Google Play

---

##  Estructura del proyecto

- `Assets/` – Código fuente, escenas y recursos
- `Packages/` – Dependencias del proyecto
- `ProjectSettings/` – Configuración del proyecto Unity

Las carpetas generadas automáticamente por Unity (`Library`, `Temp`, etc.) están excluidas mediante `.gitignore`.
